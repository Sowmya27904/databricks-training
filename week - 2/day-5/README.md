# Week 2 - Day 5: Regex Foundations & Advanced Pattern Matching

This day covers Regular Expressions (REGEX) inside SQL. You will learn anchor tags, quantifiers, continuity bounds, wildcards, and capturing groups to extract custom tokens.

## Topics Covered

- Start and end anchors (`^`, `$`)
- Character and digit matching classes (`[a-z]`, `[0-9]`, wildcards)
- Quantifiers (`+`, `{}`) and escaping sequences
- Custom token parsing using `REGEXP_SUBSTR`
- Extraction capture groups (e.g. usernames, domain components, serial codes)
