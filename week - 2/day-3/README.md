# Week 2 - Day 3: Advanced Joins & Set Operations

This day focuses on multi-table relationships. You will master standard join variations, self-referencing joins, set unions, and data aggregation across tables.

## Topics Covered

- Relational joins (`INNER`, `LEFT`, `RIGHT`)
- Emulating MySQL `FULL OUTER JOIN` using `UNION`
- Self-joins for hierarchical models (Employee-Manager)
- Grouping and aggregation across multiple tables
- Mismatch auditing and NULL replacement (`COALESCE`, `IFNULL`)

## Files in This Folder

- `practice_queries.txt`  
  Contains 30 joins and set operations exercises with schema definitions and mock data inserts.

- `solution_queries.sql`  
  Contains SQL solutions for all 30 join exercises.
