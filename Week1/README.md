Week 1 - Day 1

SQL Schema Creation & Basic Query Practice

---

📌 Overview

On Day 1, I focused on creating a structured database schema and practicing fundamental SQL queries. This forms the foundation for understanding data retrieval, filtering, and basic operations.

---

🗂️ Files Included

1. "schema.sql"

- Contains SQL statements to:
  - Create tables:
    - Department
    - Employee
    - Project
  - Define primary and foreign keys
  - Insert sample data into all tables
- Also includes additional data to test edge cases such as:
  - Employees without a department
  - Projects without a department

---

2. "q1_to_q35.sql"

- Contains solutions for SQL queries (Q1–Q35)
- Covers:
  - Basic SELECT statements
  - Filtering using WHERE
  - String pattern matching (LIKE)
  - Date-based queries
  - Aggregate functions (SUM, AVG, COUNT)
  - GROUP BY and HAVING
  - ORDER BY sorting

---

3. "output.txt" (or screenshots)

- Stores outputs of executed queries
- Helps in verifying correctness of SQL queries

🧠 Topics Covered

- Grouping & Sorting

Day 2

---

📌 Overview

On Day 2, I focused on advanced SQL concepts including joins, subqueries, and complex query combinations. These concepts are essential for working with multiple tables and solving real-world data problems.

---

1. "q36_to_q65.sql"

- Contains SQL queries from Q36 to Q65

  - JOIN operations (INNER JOIN, LEFT JOIN)

2. "output.txt" (or screenshots)

---
